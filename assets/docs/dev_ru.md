# About extend

### Shortcuts: 

#### Global
- [M] - скрывается/отображает панель MAP
- [F] - скрывается/отображает поисковой элемент MAP
- [С] - переключает отображения MAP (glyphs / components)


#### In Canvas
- [delete] / [backspace] - удаляет элемент с полотна


#### In Glyph Preview
- [S] - открывает информацию о глифе / Alt: [I]
- [F] - режим редактирования дельт (f-type) ( из условия fenerative font)
- [G] - режим редактирования Guidelines
- [H] - режим редактирования точек (horner) / Alt: [P],[B] 


// При условии процедурного подхода и активного режима (f-type)
- [shift] (toggle) - снэпинг к констрейнам из условия, что они присуствуют в рецепте

// В режиме редактирования точек (horner)
- [ctrl + z] - undo
- [ctrl + y] - redo
- [ctrl + v] - full rewrite paste from SVG buffer

### Структура: 
// В процессе

```
fe\ ← Корневая директория
 │
 ├─ index.html   ← (DOC)
 ├─ style.css    ← (стили страницы)
 ├─ main.js      ← (логика взаимодействия с пользователем / бизнес логика)
 │
 ├─ assets\      ← Сторонние файлы на странице
 │     ├─ docs\..
 │     ├─ img\..
 │     └─ svg\..
 │
 ├─ libs\        ← Сторонние миницифицированные библиотеки
 │
 ├─ core\        ← отдельные js модули + классы
 │   │      
 │   ├─ init.js  ← модуль отвечающий за логику при загрузке 
 │   ├─ utils.js ← модуль содержащий вспомогательные функции
 │   │
 │   ├─ ProcedureGlyph.js  ← внутренние классы
 │   └─ GlyphRecipes.js    ← готовые рецепты (шаблоны)
 │
 ├─ dev\              ← Папка содержащие файлы разработки ( присутствует в git ветке fev\ )
 │   │
 │   ├─ !backup\      ← Full Backups folder
 │   │    ├─ ...\
 │   │    └─ ...\   
 │   │
 │   ├─ comments\     ← Прямые комментарии
 │   ├─ debug\        ← Файлы используемые для дебага
 │   ├─ design\       ← Файлы дизайна (psd, ai, etc...)
 │   ├─ docs\         ← Файлы под документацию
 │   └─ js\           ← js temp folder: прототипы / сегментированные по логике блоки
 │
 ├─ README.md   ← вводная информация под github
 └─ LICENSE     ← шаблон лицензии от github

```
