# About extend

## Shortcuts: 

- [space] - переключает режимы Canvas (free / align on spline)

- [M] - скрывается/отображает панель MAP
- [F] - скрывается/отображает поисковой элемент MAP
- [С] - переключает отображения MAP (glyphs / components)

#### In Global Canvas
- [ctrl + A] - выделяет все элементы на полотне
- [delete] / [backspace] - удаляет элемент с полотна


#### In Glyph Preview
- [S] - открывает информацию о глифе / Alt: [I]
- [F] - режим редактирования дельт (f-type) ( из условия fenerative font)
- [G] - режим редактирования Guidelines
- [H] - режим редактирования точек (horner) / Alt: [P],[B] 


// При условии процедурного подхода и активного режима (f-type)
- [shift] (toggle) - снэпинг к констрейнам из условия, что они присуствуют в рецепте

// В режиме редактирования точек (horner)
- [ctrl + z] - undo
- [ctrl + y] - redo
- [ctrl + v] - full rewrite paste from SVG buffer

### ProcedureGlyph.js

- #### ShapeData
Определяет "сплайн" / "кривые" в проекте
- нужен для рисования кривых типа bspline / bezier-cubic
- epts хранит регулирующие точки
- pathPoints хранит набор равномерно распределённых точек вдоль кривой

<br>

- #### PenLine
Определяет элементы управления StemLine (промежуточные/опорные контроллеры)

<br>

- #### StemLine
Определяет процедурный штрих
- логика constrains реализована здесь через гетеры

<br>

- #### ProcedureGlyph
Определяет процедурный глиф 
- хранит рецепт
- имеет метод конвертации в openType glyph path (с нормализацией через paper.js)
